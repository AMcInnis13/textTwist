    Create src/main/java/cs120/lab06/TwistController.java with the following specification:

    Package: cs120.lab06
    Imports: java.util.ArrayList, java.util.Collections, java.util.HashSet, java.util.List, java.util.Random (no javafx imports — zero tolerance per D-13)

    Fields (all private — per MODEL-01):
      - int score        (starts at 0 per D-14)
      - int levelScore   (starts at 0 per D-14)
      - int level        (starts at 1 per D-14)
      - String targetWord
      - WordDictionary myDictionary
      - HashSet&lt;String&gt; guessedWords   (per D-11; tracks words already found this episode)

    Constructor:
      TwistController(String folderPath) — instantiates myDictionary = new WordDictionary(folderPath), sets level=1, score=0, levelScore=0

    Method beginEpisode(int letterCount):
      - Calls myDictionary.initialize(letterCount) to load the word file for that letter count
      - Sets targetWord = myDictionary.randomWord(letterCount)
      - Resets levelScore = 0
      - Clears guessedWords = new HashSet&lt;&gt;()
      - Does NOT reset score (score persists across episodes)
      - Does NOT increment level (level management is PrimaryController's responsibility in Phase 3)

    Method checkGuessWord(String wrd):
      - If wrd is null or empty, return 0
      - Normalize: String upper = wrd.toUpperCase()
      - If guessedWords.contains(upper), return 0 (duplicate per D-11)
      - If !myDictionary.isValidWord(upper), return 0 (invalid)
      - Otherwise: int n = upper.length(); int points = n * n * 10
      - Add upper to guessedWords
      - Add points to both score and levelScore
      - Return points
      - Note: does NOT check formability (letter subset) per D-10

    Method shuffleLetters():
      - Convert targetWord characters to List&lt;String&gt;: iterate chars, add String.valueOf(c) for each
      - Shuffle in-place: Collections.shuffle(list, new Random())
      - Return the shuffled List&lt;String&gt;

    Getters (public, for test and PrimaryController access):
      - int getScore()
      - int getLevelScore()
      - int getLevel()
      - String getTargetWord()

    No setters needed for Phase 1. No JavaFX imports. No static fields.
  </action>
