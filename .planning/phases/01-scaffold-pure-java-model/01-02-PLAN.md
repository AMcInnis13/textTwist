  </read_first>
  <action>
    Create src/main/java/cs120/lab06/WordDictionary.java with the following specification:

    Package: cs120.lab06
    Imports: java.io.File, java.util.HashSet, java.util.ArrayList, java.util.Random, java.util.Scanner (no javafx imports — zero tolerance)

    Fields:
      - private String folderPath  (set in constructor)
      - private HashSet&lt;String&gt; words  (populated by initialize())

    Constructor:
      WordDictionary(String folderPath) — stores folderPath, initializes words as new HashSet&lt;&gt;()

    Method initialize(int letterCount):
      - Build file path: folderPath + "/" + letterCount + ".txt"
      - Open via new File(path) and wrap with Scanner
      - Read first line as int wordCount (the file's declared count)
      - Read remaining lines into words HashSet; call trim() on each line before adding
      - Clear the existing words HashSet before loading (allows re-initialization for different letter counts)
      - Do NOT use getResourceAsStream — file is at project root, not on classpath

    Method isValidWord(String wrd):
      - Normalize: return words.contains(wrd.toUpperCase())
      - Return false if wrd is null or empty

    Method randomWord(int n):
      - Convert words to new ArrayList&lt;String&gt;(words)
      - Return list.get(new Random().nextInt(list.size()))
      - Note: the n parameter is documented as "the length of words in the loaded set" (caller must have called initialize(n) first). The method does NOT re-load the file; it returns a random word from the currently loaded set.

    No other public methods needed for Phase 1. No JavaFX imports. No static fields.
  </action>
