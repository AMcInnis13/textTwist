# textTwist
Text twist extra credit assignment for devOps class

I used Claude code on a high thinking sonnet model, intellij as an ediitor, with scenebuilder to check progress. The two plugins i used were GSD (https://github.com/gsd-build/get-shit-done/blob/main/README.md) and Superpowers (https://github.com/obra/superpowers)

# reflection 
I ran into few problems. I ran out of tokens twice in the 5 hour context window (i did not work on it for 10 hours, in fact i did other work while doing this.) Using the GSD plugin, was helpful as it had 6 phases which i understood and could loop through easily, it just ate a bunch of tokens. It worked well, and explained everything it was doing and its thought process.
